# Getting Started

Welcome to the platform knowledge base. This documentation will help you use the platform effectively.

## Finding help

- Use the **Docs** link in the sidebar (when in the admin panel) to open this knowledge base.
- Use the search at the top to find articles by title or content.
- Contextual help may appear on specific resources or pages.

## Managing the platform

The following guides cover the main areas of the platform:

- **[Managing Workspaces](workspaces)** — Create and edit workspaces, add members, and switch between workspaces.
- **[Managing Projects](projects)** — Set up projects, status, dates, and integrations (GitHub, Slack, Telegram, WhatsApp).
- **[Managing Boards](boards)** — Define workflow columns (e.g. Pending, Ongoing, Review, Completed) for your workspace.
- **[Managing Tasks](tasks)** — Create tasks, assign users, use checklists, and move tasks between boards.
